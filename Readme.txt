Show your reader how much time they need to read and how many words in a post.

== Description ==

If you are a blogger or writer and using WordPress as your website CMS, then WP Word & Reading Time Counter plugin is for you. It will tells you exactly how many words you’ve written on your site and how long it will take to read. Reading Time options will shows the estimated reading time of the post, in seconds or minutes.

Features:

*   Post Word Count
*   Post Reading Time
*   Easy to Configure
*   Easy to use
*   Compitable with Latest version

N.B: If you want more options in this simpel plugin, please [contact with me](https://wprashed.com "Rashed - WordPress Developer")

== Installation ==

Installing 'Word & Reading Time Counter' plugin is easy as like dirinking tea at home. :) Just follow these simple steps:

1. Go to Add new Plugin from your WordPress admin and search for: Word & Reading Time Counter
1. Click on install and activate after successfully installed.
1. Thats it.
